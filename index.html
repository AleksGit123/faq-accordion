FAQs

  • What is Fronted Mentor, and how will it help me?

    • Frontend Mentor offers realistc coding challenges to help developers improve their fronted coding skills with projects in HTML, CSS, and JavaScript. It’s suitable for all levels and ideal for porfolio building.

  • Is Frontend Mentori free?

    • Yes, Frontend Mentor has both free and paid options. The free version provides access to some challenges with JPG design previews for practice. The PRO version includes Figma files, advanced challenges, and exclusive community access.

  • Can I use Frontend Mentor projects in my portfolio?

    • Yes, you can use Frontend Mentor projects in your portfolio. The platform encourages developers to showcase their completed challenges as part of their work. These projects help demonstrate real-world skills in HTML, CSS, and JavaScript. Make sure to add your own touches to stand out.

  • How can I get help if I’m stuck on challenge?

    • If you're stuck on a Frontend Mentor challenge, there are several ways to get help. First, check the Frontend Mentor community forums, where developers share tips and solutions. You can also ask for feedback by posting your code on Discord or GitHub. Reviewing the challenge's README file and design files might help clarify requirements.
